Coding

What

Coding is a shortened term for computer programming, which involves writing a set of instructions for a computer to carry out to complete a task. There are many different types of programming languages to choose from. Different languages are used for different things. For example, this app is written in a language for making easy-to-use mobile web pages called jQuery Mobile. Many data scientists use Python or R to crunch numbers. Websites are made in HTML and CSS.

Why

If you like to solve problems and create new things, coding is a great hobby for anyone. It's also a very employable skill. Many jobs that you wouldn't think of as computer jobs consider coding skills a plus in today's market. The number of coding jobs in America is expected to grow by 30% by 2020.

How

Learning to code takes time. It's best to concentrate on one language completely before moving on to other languages. Many books are available to learn to code and many interactive sites to teach coding are free. If you want to learn to code, consider adding a one-hour block five days a week to your schedule.
